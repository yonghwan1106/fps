## v3.0.0 - Rivals 스타일 게임플레이 오버홀
**작업일시**: 2025년 12월 17일 (화) 14:24

### 무기 시스템 대폭 확장
- 8종 무기 추가
  - Pulse Rifle (펄스 라이플)
  - Phantom AR (팬텀 돌격소총)
  - Viper SMG (바이퍼 기관단총)
  - Neon Shotgun (네온 산탄총)
  - Ion Sniper (이온 저격총)
  - Plasma LMG (플라즈마 경기관총)
  - Arc DMR (아크 지정사수소총)
  - Volt Pistol (볼트 권총)
- ADS (조준경) 시스템 구현
- 헤드샷 시스템 추가
- 거리별 데미지 감소 시스템

### 이동 시스템
- 슬라이딩 기능
- 대시 기능 (2회 충전식)
- 벽타기 (Wall Running)

### 킬스트릭 보상
- 3/5/7/10/15킬 달성 시 특별 보상 지급

### 어빌리티 시스템 (4종)
- **Blink**: 순간이동
- **Overcharge**: 과충전 (공격력 증가)
- **Phase Shift**: 위상 전환 (무적)
- **Scanner**: 적 위치 스캔

### 패시브 퍽 시스템
- 8종 패시브 퍽 추가
- 퍽 선택 UI 구현

### 게임 모드 (3종)
- **Survival**: 생존 모드
- **Wave Assault**: 웨이브 어설트 (보스 포함)
- **Time Attack**: 타임 어택

### 보스 시스템
- Boss 적 클래스 추가
- 특수 공격 패턴 구현

### UI/UX 개선
- 새로운 HUD 디자인
- 로드아웃 선택 화면
- 어빌리티/퍽 카드 UI
- 모바일 컨트롤 (슬라이드, 대시, 어빌리티 버튼)

**변경 파일**: `game.js`, `index.html`, `style.css` (+3,689줄)

### 난이도 설정
- 쉬움 / 보통 / 어려움 3단계 난이도 선택 가능

### 밸런스 조정
- 구체별 데미지 차등화
  - 파랑 구체: 20 데미지
  - 빨강 구체: 15 데미지
  - 노랑 구체: 10 데미지
- 총알 개수 조정
  - 피스톨: 20발
  - 라이플: 50발
  - 샷건: 10발

### 맵 개선
- 맵 전체적으로 밝게 조정
- 불필요한 벽 제거로 시야 확보

**변경 파일**: `Game.tsx`, `Menu.tsx`, `Scene.ts`, `Target.ts`, `gameStore.ts`

## v1.0.0 - FPS Arena 최초 구현
**작업일시**: 2025년 12월 15일 (일) 08:15

### 핵심 기능
- Three.js 기반 3D FPS 게임 구현
- 1인칭 시점 카메라 시스템
- 마우스 포인터 락 기능

### 무기 시스템 (3종)
- 권총 (Pistol)
- 라이플 (Rifle)
- 샷건 (Shotgun)

### 오디오 시스템
- Web Audio API 기반 사운드 효과
- 총성, 재장전, 피격 사운드

### 리더보드
- 로컬스토리지 기반 점수 저장
- 최고 기록 표시

### UI/HUD
- 반응형 HUD 디자인
- 체력바, 탄약 표시, 점수 표시
- 메인 메뉴 / 게임 오버 화면
